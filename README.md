# Dr. Mohammad Abu Kausar — Academic Website

Ready for GitHub Pages.

## Publish
1. Create a public repository named `mohammadabukausar.github.io`.
2. Upload all files and folders from this package to the repository root.
3. Open Settings → Pages and deploy from the `main` branch, root folder.
4. The site will appear at your GitHub Pages address.
